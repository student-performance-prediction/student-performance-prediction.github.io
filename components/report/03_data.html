    The data for this project is available for download from the UCI Machine Learning Repository http://archive.ics.uci.edu/ml/machine-learning-databases/00320/

    The zip file downloaded from the above link consists of two files student-por.csv and student-mat.csv. These files represent the student data collected for the Portuguese language and Mathematics class.
